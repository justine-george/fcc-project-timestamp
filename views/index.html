Timestamp Microservice

API Usage:

  • /api/2023-01-01 (Converting a Date String to Timestamp)
  • /api/1672531200000 (Converting a Timestamp to Date String)
  • /api/ (Getting the Current Timestamp and Date)

Output:

{"unix": 1672531200000, "utc": "Sun, 01 Jan 2023 00:00:00 GMT"} (For both Date String to Timestamp and Timestamp to Date String conversions)

{"unix": <current_unix_timestamp>, "utc": "<current_utc_date>"} (For Getting the Current Timestamp and Date)
